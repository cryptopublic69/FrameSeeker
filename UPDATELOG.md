# 更新日志

本文记录 FrameSeeker 的主要功能与接口变更。

## 2026-07-15 — 初始版本

### 多模态检索核心

- 集成 SigLIP 2 Giant 1B Patch16 384、DINOv3 ViT-H+/16 与 pHash，分别用于语义检索、视觉结构检索和重复图片识别。
- 使用 Qdrant 1.18 持久化语义向量、视觉向量、图片指纹及资产元数据，默认使用 `framefinder_assets_v3` 集合。
- 提供文字搜图和以图搜图能力，并按文件内容 SHA-256 复用已计算向量，减少重复推理。
- 提供轻量健康检查和缓存统计，支持 CUDA/CPU 运行状态展示。

### ImageLib 4.x 正式 API

- 新增带 `X-FrameFinder-Key` 鉴权的 `/api/v1` 接口，并提供统一错误结构、请求 ID、速率限制提示与 OpenAPI 描述。
- 支持能力查询、资产新增或更新、单项及批量状态对账、单项及批量删除、元数据更新和资料库资产清单。
- 支持按资料库隔离的全库文字搜索与已入库资产相似搜索，以及标签、虚拟目录、存储根目录和媒体类型等元数据过滤。
- 使用 `(library_id, asset_id)` 映射稳定 UUIDv5 Point ID，并通过 `client_revision`、内容哈希和元数据哈希处理幂等更新与冲突。
- 支持稳定分页游标和搜索会话过期检测；当前明确不支持异步索引任务和临时上传图片搜索。

### Web 工作台

- 新增基于 React、Next.js、vinext 与 Cloudflare Worker 的响应式中文检索界面。
- 支持一次选择最多 200 张候选图片、文字/图片检索模式切换、参考图选择、结果评分、重复标记和缓存命中统计。
- 提供 Qdrant、模型和运行设备状态展示，并适配桌面端与移动端布局。

### 部署与运维

- 新增 Docker Compose Qdrant 服务，以及 Windows 一键启动、停止和局域网防火墙配置脚本。
- 启动流程可自动检测局域网 IPv4、生成并保护本地 API 密钥、检查 Qdrant/后端/前端健康状态并打开 Web 工作台。
- 提供可复现的 Python 与 Node.js 依赖锁定、OpenAPI 导出文件和 Cloudflare Sites 托管配置。
- 将 API 密钥、虚拟环境、模型缓存、构建产物、依赖目录和本机运行状态排除在版本控制之外。

### 测试与文档

- 新增不加载大型模型权重的 ImageLib API 契约测试，覆盖鉴权、OpenAPI、资产隔离、版本控制、过滤分页、元数据、清单和幂等删除。
- 新增模型烟雾测试与前端构建/渲染测试。
- 新增 README、局域网资产管理对接文档和 ImageLib 4.x 前端联调交接文档。
